# sao-simple-web

> simple web SAO generator

## Usage

Install [SAO](https://github.com/saojs/sao) first.

```bash
yarn global add sao
# or
npm i -g sao
```

### From npm

```bash
sao simple-web my-project
```

### From git

```bash
sao lpreterite/sao-simple-web my-project
```
